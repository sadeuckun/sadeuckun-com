Accessibility-Focused Staff Panel

This was an accessibility-focused staff panel built for a large public-sector related organization. The goal was to make a complex internal system significantly easier to use for visually impaired users.

Project details. The platform was developed with Vue.js on the frontend and Symfony on the backend. It included custom keyboard shortcuts, tabindex logic and non-visual interaction patterns designed to support smoother navigation for screen reader and keyboard-first users.

My contribution. I took ownership of the frontend improvements, refactored the existing store-controller structure, improved code modularity and worked on accessibility-focused interaction design instead of treating accessibility as a superficial checklist.

I implemented keyboard navigation improvements, custom shortcuts, focus management and ARIA-minded refinements that made the interface more fluid and predictable for users relying on assistive workflows.

To improve reliability, I also added and maintained Nightwatch end-to-end tests, helping the team validate critical user flows with more confidence.
