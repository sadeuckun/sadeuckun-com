Company Management Platform

Company Management Platform is a central business operations system that brings tasks, leave management, offers and company tracking into a single self-hosted product.

Project details. The platform was designed to replace a fragmented workflow spread across tools like Trello, Jira, Clockify and Excel. It combines project and task management, time tracking, offers, service catalog flows, leave approval processes, company and team tracking, password management and operational reminders in one place.

It includes drag-and-drop task management, Kanban-style workflows, Clockify integration, PDF offer generation, leave request flows, domain and server expiry tracking, employee accounting modules and notification flows through email and WhatsApp.

My contribution. I led the project as a full-stack developer and built it from scratch using Next.js App Router, MongoDB and related tooling. I designed the SSR and SSG architecture, created the database schema and API routes, and implemented the main operational modules end to end.

I also built real-time features with Socket.io, automated PDF generation with Puppeteer, cron-based reminders, OTP authentication, IP restrictions, encrypted credential storage and backend performance improvements through aggregation, indexing and query optimization. The system was deployed on Linux with PM2 and integrated into a CI/CD pipeline.
