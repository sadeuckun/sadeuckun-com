BioVet Insight

BioVet Insight is an AI-assisted decision support platform built for veterinary clinics. The product combines patient management, laboratory tracking and operational workflows in a single system.

Project details. The goal was to digitize clinic operations while also helping veterinarians reach information faster and interpret clinical context more effectively. The platform was built with Go and React and designed with scalability in mind.

My contribution. I worked on application flows, API integrations, data processing and the broader system architecture needed to keep the product sustainable as it grew.

A major part of the platform was an LLM + RAG based decision support workflow that evaluated patient history, laboratory findings and related records to support faster, more contextual analysis.

This project represents the kind of work I enjoy most: building software that turns complex domain knowledge into faster, more structured and more useful workflows.
