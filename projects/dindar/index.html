Dindar

Dindar is a mobile guidance and information app built around prayer times, Quran reading, dua management and daily personal use flows.

Project details. The product was developed with FastAPI and React Native, with a focus on mobile usability, API integration quality and a backend structure that could support long-term growth.

My contribution. I worked on the mobile and backend sides together, shaping a product experience that felt responsive, practical and technically sustainable.

One of the most interesting parts of the app was an AI-assisted helper system built around religious sources, designed to answer user questions with more contextual responses supported by relevant references.

Dindar highlights the combination of mobile product development, AI-assisted information access and scalable backend thinking in a single project.
